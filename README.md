# NSW Leave Calculator PWA

A mobile-friendly, offline-capable calculator for Annual Leave and Long Service Leave (LSL) specifically designed for New South Wales, Australia.

## Features
- **NSW LSL Compliance:** Uses the 8.6667 weeks per 10 years formula (NSW LSL Act 1955).
- **Casual Support:** Toggles between permanent and casual entitlements.
- **PWA Ready:** Installable on iOS/Android home screens and works offline.
- **Privacy First:** All calculations are done locally in your browser. No data is stored or transmitted.

## Deployment
This repository is configured for GitHub Pages.
1. Go to **Settings > Pages**.
2. Select the `main` branch.
3. Your app will be live at `https://edmundchow.github.io/nsw-leave-calc/`.
